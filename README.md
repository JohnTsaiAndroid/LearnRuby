##如何使用
* 1.将代码clone下来
* 2.cd至LearnRuby相应目录下
* 3.ruby XX.rb即可（其中ruby XX.rb true会将源代码打印出来，但过滤掉注释，ruby XX.rb false则不会过滤，不填写true
或者false则只输出结果不打印源代码和注释）
